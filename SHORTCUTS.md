# TVRL Keyboard Shortcuts

This document lists all the keyboard shortcuts available in the Terminal Visual Curl (TVRL) application.

## Navigation

| Shortcut          | Action                                |
|-------------------|---------------------------------------|
| Tab / Right Arrow | Move to next tab                      |
| Shift+Tab / Left Arrow | Move to previous tab             |
| Up Arrow          | Navigate up in templates list         |
| Down Arrow        | Navigate down in templates list       |

## Application Controls

| Shortcut          | Action                                |
|-------------------|---------------------------------------|
| Ctrl+Q            | Quit application                      |
| F1                | Show help / Hide help                 |
| Esc               | Cancel current operation / Close dialog |

## Panel Controls

| Shortcut          | Action                                |
|-------------------|---------------------------------------|
| Ctrl+T            | Toggle templates panel                |
| Ctrl+E            | Toggle environments panel             |
| Ctrl+H            | Toggle history panel                  |

## Command Operations

| Shortcut          | Action                                |
|-------------------|---------------------------------------|
| Ctrl+N            | New command                           |
| Ctrl+S            | Save template                         |
| Ctrl+Enter        | Execute command                       |
| Ctrl+C            | Copy command to clipboard             |

## Tab Navigation

The application has the following tabs that you can navigate between using Tab/Shift+Tab or Left/Right arrows:

1. URL - Configure the URL and HTTP method
2. Headers - Manage request headers
3. Body - Configure the request body
4. Curl Options - Set curl command options

## Mouse Support

While the application is primarily keyboard-driven, some mouse operations are supported:

- Click on tabs to switch between them
- Click on templates to select them
- Click on checkboxes to toggle options
